## 🗳️ راهنما

- **پیشنهاد موضوع جدید**: یک Pull Request ارسال کنید و موضوع خود را به لیست زیر اضافه کنید.
- **اعلام علاقه‌مندی برای ارائه**: چک‌باکس کنار هر موضوع را تیک بزنید(در صورت عدم دسترسی برای تغییر چک باکس، میتوانید پول ریکوئست برای تغییر وضعیت آن بدهید).
